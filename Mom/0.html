Let's do it!

To use Python, you need to know some basic commands in the Terminal.

  1. First, click on your Launchpad (the rocketship in your Dock), go into the Other folder, and click Terminal. A blank screen with some text should come up.
  2. First command: open. In your terminal, type a line that looks like this: Jills-MBP:~ Jill Simon$ open -a Safari
    3. Don't worry if the first part doesn't look exactly like I typed, as long as it looks similar it's fine. Press enter to submit the command. If this window came up, you did it right!
  3. Now try:Jills-MBP:~ Jill Simon$ open -a "Sublime Text 2"
    4. Did a text editor come up? If so, great!
    From now on, I will only type the $ and everything afterward, because everything before is done for you and doesn't really matter right now. You don't need to type the opening $ though, it's already there.
  4. So, the open command opens things!
    What the -a does is specify that you're opening an application. That way, your computer doesn't have to look for some file named Sublime Text 2 or Safari. Also, I used quotes around the Sublime Text 2 so it treats it as one word, not 3. It is a concept you will learn more about tomorrow.
  5. Next command: ls. Type$ lsIt should come up with a list of all the files in your home folder, or ~. You should recognize some of these! What ls does is lists all the files and directories, or folders, in the folder you are currently in.
  6. Type$ cd Documentsand press enter. Notice that the ~ at the beginning of the line changed to Documents. What the cd command does is changes directories on your computer. Basically, it's the same as you double-clicking on a folder in Finder: it goes into it.
  7. Type$ mkdir CodingWhat this did was make a new directory. Test that using ls the same way as step 5. The output would be everything in your Documents folder. Do you see something called Coding in there?
  8. Like in Step 6, move into the Coding directory.
  9. Last new command:$ touch helloworld.pyYou can see what this did if you use ls again. What touch does is create a file in the folder you are currently in.
    In these challenges, always use .py at the end. This tells the computer it's a python file.
  10. Finally, open can open files too. Type$ open helloworld.py -a "Sublime Text 2".The text editor should open to a blank file, but at the top of the window, you should see "helloworld.py". To break it down, you opened the file we just made in the -application Sublime Text 2, what we downloaded earlier.
